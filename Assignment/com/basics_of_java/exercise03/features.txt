Simple :
         1) Java syntax is similar to c++ so it is simple to learn if you have knowledge of c++.
         2) java removed the concept of pointers and operator overloading .

--------------------------------------------------------------------------------------------------

Secure :
        Because it does not use the pointers which makes it virus-free.

---------------------------------------------------------------------------------------------------

Portable :
          Java language is portable because we can write the program on any operating sytem(windows,linux,...) and can run 
          the generated bytecode on other operating system.

--------------------------------------------------------------------------------------------------

Robust :
         Java provides automatic garbage collector which helps to memory management that makes it robust.

----------------------------------------------------------------------------------------------------

Architecture-Neutral :
                      Java is architecture neutral because there are no implementation dependent features, for example,  the
                      size of primitive types is fixed.
----------------------------------------------------------------------------------------------------
High Performance : Java uses the JIT (Just In Time) compiler. The compiler only compiles that method which is being called.
                  
----------------------------------------------------------------------------------------------------
Distributed :
              Java distributes the code that gets easily compatible with the programming environment
----------------------------------------------------------------------------------------------------
Dynamic :
         Java loads classes on demand that's by java is dynamic language.
